# Employee Management System

A simple desktop Employee Management System built with **Python**, **CustomTkinter**, **Pillow**, and **MySQL**.  
Allows you to **login**, **add**, **update**, **delete**, and **search employee records**.

---

## Features
- Login authentication
- Add new employees
- Update existing employee records
- Delete single or all employee records
- Search employees by ID, Name, Role, Gender, Phone, or Salary

---

## Requirements

- Python 3.13
- MySQL Server (for database storage)
- Python packages:
  - customtkinter
  - pillow
  - pymysql

You can install the packages using pip:

```bash
pip install customtkinter pillow pymysql
